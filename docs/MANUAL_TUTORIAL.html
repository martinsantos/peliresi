Sistema de Trazabilidad de Residuos Peligrosos

Manual completo del sistema para la gestión integral del ciclo de vida de los manifiestos de residuos peligrosos en la Provincia de Mendoza.

🔐 Acceso al Sistema

El Dashboard Web está protegido con contraseña: mimi88

La Demo App (selector de roles) tiene acceso libre.

Actores del Sistema

Actor Descripción Casos de Uso
Administrador DGFA Supervisa y administra el sistema completo CU-A01 a CU-A15
Generador Empresa que produce residuos peligrosos CU-G01 a CU-G12
Transportista Empresa habilitada para transporte CU-T01 a CU-T11
Operador Planta de tratamiento/disposición final CU-O01 a CU-O12

⚡ Cambio de Perfil en el Menú

IMPORTANTE

El sistema permite cambiar de rol desde el menú lateral de navegación del Dashboard.

Para cambiar de perfil en el Dashboard:

  1. Acceder al Dashboard con la contraseña mimi88
  2. En el menú lateral izquierdo, localizar la sección de usuario
  3. Hacer clic en el selector de rol para cambiar entre perfiles
