Guia de Instalacion

La instalacion de SITREP sigue 3 fases. Cada fase tiene verificaciones que aseguran que el sistema quede correctamente configurado.

A. Pre-Instalacion
→
B. Wizard Interactivo
→
C. Post-Instalacion

Fase A: Pre-Instalacion

Antes de ejecutar el wizard, necesitas:

ServidorUbuntu 22.04 LTS, 2+ cores, 4GB+ RAM, 80GB+ SSD
DNSA record apuntando al IP del servidor (Cloudflare)
SSHAcceso root al servidor
Repositoriogit clone https://github.com/martinsantos/peliresi.git
BrevoCuenta gratuita en brevo.com para envio de emails
ssh root@IP_DEL_SERVIDOR
apt update && apt upgrade -y
apt install -y curl wget git ufw fail2ban htop
ufw allow OpenSSH && ufw allow 80/tcp && ufw allow 443/tcp && ufw enable

Fase B: Wizard Interactivo (8 pasos)

Abrir SETUP.html en el navegador. El wizard guia paso a paso:

  1. Servidor Web — Verificar Nginx + SSL
  2. Base de Datos — Docker + PostgreSQL 15
  3. Backend API — Node.js 20 + PM2 + .env
  4. Frontend — React SPA (Vite build)
  5. Admin Inicial — Seed de datos base
  6. Actores Reales — Importar 1,846 generadores + 52 operadores del registro DPA
  7. Email — Brevo SMTP + SPF/DKIM/DMARC
  8. Verificacion Final — Resumen + checklist
Abrir Wizard →

Fase C: Post-Instalacion

1Cambiar password adminLogin → Perfil → Cambiar contrasena
2Backup automaticocrontab -e → 0 3 * * * /opt/scripts-cicd/backup_sitrep.sh
3Health check*/5 * * * * /opt/scripts-cicd/healthcheck.sh
4Activar emailsDISABLE_EMAILS=false en .env + pm2 restart
5Smoke testbash backend/tests/smoke-test.sh
6Accederhttps://tudominio.com/login
